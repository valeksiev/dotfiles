### Setup
- install `homebrew`
- install `iterm2`
- install `zhs`
- install `python3`
- install `vim` 
- install `the_silver_searcher`, `ag`
- ensure `vim --version | ag python` returns `python3`
- link `~/.zshrc` to `./.zhrc`
- link `~/.vimrc` to `./.vimrc`
- link `~/.vim` to `./.vim`
- start vim && `Bundle Install`
